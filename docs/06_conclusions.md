# ✅ Conclusão

A análise exploratória permitiu identificar padrões relevantes no comportamento das variáveis clínicas.

## Principais insights

* maior tamanho tumoral está associado a pior prognóstico
* maior número de linfonodos positivos reduz a sobrevida
* estágios avançados apresentam menor tempo de sobrevivência
* pacientes com status hormonal positivo tendem a melhor prognóstico
