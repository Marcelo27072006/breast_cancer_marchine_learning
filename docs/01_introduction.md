# 📌 Introdução

Este projeto tem como objetivo realizar uma **Análise Exploratória de Dados (EDA)** em um dataset clínico de pacientes com câncer de mama.

O projeto tem como objetivo o desenvolvimento de uma solução baseada em Aprendizado de Máquina para análise de dados clínicos, com foco na previsão do agravamento do câncer de mama. A proposta consiste em criar um modelo capaz de identificar padrões nos dados e prever a evolução do estado do paciente, classificando cenários de melhora, estabilidade ou piora clínica ao longo do tempo. 
O foco principal é investigar padrões que possam influenciar o **tempo de sobrevivência dos pacientes**, considerando variáveis como:

* idade
* tamanho do tumor
* estágio tumoral
* linfonodos positivos
* status hormonal
* estado civil

Além dessas, o dataset contém outras variáveis relevantes que serão detalhadas na seção de descrição dos dados.

A EDA foi utilizada como etapa inicial para compreender o comportamento dos dados e apoiar futuras análises preditivas.

A implementação do projeto mostrou-se tecnicamente viável, considerando os seguintes fatores:

Dataset Internacional estruturado analisado: https://www.kaggle.com/datasets/reihanenamdari/breast-cancer

As ferramentas empregadas são gratuitas e amplamente suportadas

A análise pode ser executada em máquinas com recursos computacionais básicos, sem necessidade de infraestrutura complexa para a fase inicial do projeto

Além disso, as bibliotecas utilizadas possuem ampla documentação, o que facilita a manutenção e evolução do projeto.
