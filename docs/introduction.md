# 📌 Introdução

Este projeto tem como objetivo realizar uma **Análise Exploratória de Dados (EDA)** em um dataset clínico de pacientes com câncer de mama.

O foco principal é investigar padrões que possam influenciar o **tempo de sobrevivência dos pacientes**, considerando variáveis como:

* idade
* tamanho do tumor
* estágio tumoral
* linfonodos positivos
* status hormonal
* estado civil

A EDA foi utilizada como etapa inicial para compreender o comportamento dos dados e apoiar futuras análises preditivas.
